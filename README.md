# Calendar-App
A MERN calendar app with event management and filtering capabilities.

## Table of contents
* [Technologies](#technologies)
* [Setup](#setup)

## Technologies
Project is created with:
* JavaScript (ES6)
* HTML5
* CSS3
* React
* Vite
* Bootstrap
* React-Big-Calendar
* React Redux

## Setup
To run this project, install it locally using npm:

```
$ npm install
$ npm run dev
```
